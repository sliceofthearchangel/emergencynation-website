Privacy

Apps

Bitmessage is the currently used app for chatting

Email

protonmail is an encrypted email service but any will do. they are generally free. it is recommended that you start with a new email when coming to this community so that you do not attract any unwanted attention.

Internet

1.1.1.1 is a dns to resolve the urls that you type into your browser. it is generally a good idea to have it on your router as well as any device that is mobile / not a desktop computer. configure on router and pc or configure on mac. there are also android and iphone apps to do the same

Browser

brave is a good browser and you can tip creators anonymously for the content they put online. it has a privacy mode with tor in it. tor itself is also a good choice. it is strongly advised that you use a privacy oriented browser like brave as they take out a few things that can help other sites track that you have visited this one like referrer strings. for example try clicking on the bitmessage link in both brave and chrome and then open up the dev tools in each one. brave shows an empty string. chrome shows this site's url.

Password Management

1password is a solid app for maintaining control over your passwords and generating long, secure strings in an easy interface. 2fa (which we always recommend) is also available to setup in the app. Other managers like lastpass are good as well.

Monitoring

it is a good idea to make sure that when a breech occurs you are not succeptable to the fallout. haveibeenpwned is a solid service that monitors the web for claims of credentials posted by hackers after looting a db or service.
